# Online Shopping API Testing (Basic)

## Project Type
API Testing using Postman

## Description
This project contains API testing for a simple Online Shopping System API.
The focus of this project is authentication, role-based access, and product endpoints.

## Tested Endpoints

### Authentication
- POST /register
- POST /login

### User APIs
- GET /profile
- GET /user-data

### Admin APIs
- GET /admin-data
- POST /products (Admin only)

### Products
- GET /products

## Testing Activities
- Created Postman collection
- Used environment variables for token handling
- Verified status codes and responses
- Tested role-based authorization
- Performed positive and negative testing

## Tools Used
- Postman
- Swagger API Documentation
